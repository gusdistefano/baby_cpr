Detailed Description

ESP-IDF Passive Buzzer Driver.

ESP-IDF Buzzer Melodies.

Driver to control a passive buzzer using ESP-IDF. It provides functions to initialize the buzzer and play tones with or without volume control.

Plays music on the buzzer with a preconfigured volume.

Similar to play_music, but the volume is not specified as a parameter. Instead, it uses the volume configured through menuconfig. This is useful for cases where consistent volume levels are desired without manual adjustment.
